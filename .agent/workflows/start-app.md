## Starting the App

// turbo-all

1. Navigate to project root

2. Start the Flet GUI:
```bash
python3 gui/app.py
```

## Alternative: CLI Mode

1. Run the CLI interface:
```bash
python3 cli.py
```

2. Available CLI commands:
- `scrape` - Run scraping cycle
- `discover` - Find new sources
- `search "query"` - Search articles
- `stats` - Show statistics
