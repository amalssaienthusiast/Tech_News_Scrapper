# Start PyQt6 Application

// turbo-all

1. Activate the virtual environment (if applicable)

2. Start the PyQt6 application
```bash
python -m gui_qt.app_qt
```

The application will launch with:
- Tokyo Night dark theme
- Live article feed with tech scores
- Storage mode selector (Ephemeral/Hybrid/Persistent)
- Search functionality
- Quantum mode toggle
